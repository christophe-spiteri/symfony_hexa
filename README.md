
# Symfony Hexa

Symfony 6 préparé pour faire de l'héxa

- Symfony 6
- Web pack encore


## Installation

Cloner le project

```bash
composer install
npm install
npm run build
php bin/console make:migration
php bin/console doctrine:migrations:migrate
php bin/console d:f:l
```
