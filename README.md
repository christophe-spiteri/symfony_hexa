
# Symfony Hexa

Symfony 6.1 préparé pour faire de l'héxa

- Symfony 6.1
- Web pack encore
- User login

## Installation

Cloner le project

```bash
composer install
npm install
npm run build
php bin/console make:migration
php bin/console doctrine:migrations:migrate
php bin/console d:f:l
```
